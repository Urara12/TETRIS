 2. [pyxel](https://github.com/kitao/pyxel/blob/master/README.ja.md) をダウンロード

## 操作の方法
|key|操作|
|---|-----|
|→|落ちてくるブロックを右に動かします|
|←|落ちてくるブロックを左に動かします|
|Space|落ちてくるブロックを一気に下に落とします|
|Enter|落ちてきているブロックの向きを変えます|
